# Contributing to Get Elected

Thank you for your interest in making this skill better. Every contribution helps more people run legally compliant, strategically sound campaigns.

## How to Contribute

### Add a State

The highest-impact contribution is adding campaign finance coverage for a new state. See `states/_state-template.md` for the pattern and `states/_state-index.md` for which states need coverage.

**Requirements for state files:**
- Every compliance claim must cite a real statute, regulation, or official agency source
- Include a `last_verified` date at the top
- Note CPI adjustment schedules where applicable
- End with the educational disclaimer

### Improve Existing Content

- **Verify and update** contribution limits, filing deadlines, and agency contact info (these change)
- **Add edge cases** and practical guidance based on real campaign experience
- **Fix errors** — if you find something wrong, please open an issue or PR immediately

### Add New Modules

If you've identified a gap, open an issue first to discuss scope, then submit a PR following the existing file patterns.

## Standards

- **Nonpartisan.** This skill serves candidates of all parties. Content must be politically neutral.
- **No invented law.** Every legal claim needs a real citation.
- **Educational framing.** Campaign finance content must include the educational disclaimer.
- **Practical over theoretical.** Prioritize actionable outputs over academic explanations.
- **Progressive disclosure.** Keep SKILL.md under 500 lines. Use reference files for depth.

## Submitting Changes

1. Fork the repo
2. Create a branch: `git checkout -b state/virginia` or `fix/missouri-limits`
3. Make your changes
4. Update `references/update-log.md` with what changed
5. Update `states/_state-index.md` if you added state coverage
6. Submit a PR with a clear description of what changed and why

## Code of Conduct

This project exists to support democracy. Contributors are expected to engage respectfully regardless of political affiliation. Partisan advocacy, personal attacks, and misinformation are not welcome.
